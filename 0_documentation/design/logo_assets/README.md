# Brand logo assets — Tuned Cycling

**Default mark for UI:** [`logo_transparent_bg_noshadow.svg`](logo_transparent_bg_noshadow.svg)  
Use in top bar / profile pill / auth when a logo is needed. Prefer SVG; PNG fallbacks in this folder.

| File | Use |
|------|-----|
| `Logo_transparent_bg.svg` | Full export (may include padding) |
| `logo_transparent_bg_noshadow.svg` | **Primary for UI chrome** — no shadow, use in app |
| `Logo.svg` | Full logo on solid backgrounds |
| `App Icon.svg` / `App Icon Round.svg` | Favicon / PWA / round avatar fallback |
| `Light mode mockup.svg` / `Dark mode mockup.svg` | Reference only — not runtime assets |

**Brand colour in mark:** Hot Fuchsia `#FF0061` (see [`../BRIEF.md`](../BRIEF.md)).

Do not recolour or distort the logo in product UI without updating this doc.
